# Glossary

## Settlement Terms

| Term | Definition |
|------|------------|
| **Fail** | A trade that did not settle on the expected settlement date |
| **T+1** | Trade date plus one business day (standard settlement cycle) |
| **DTC** | Depository Trust Company - central securities depository |
| **NSCC** | National Securities Clearing Corporation |
| **CNS** | Continuous Net Settlement - NSCC's core netting system |

## Regulatory Terms

| Term | Definition |
|------|------------|
| **Reg SHO** | SEC regulation governing short sales and close-out requirements |
| **FTD** | Fail-to-Deliver - seller's failure to deliver securities |
| **FTR** | Fail-to-Receive - buyer's failure to receive securities |
| **Threshold Security** | Security with significant FTD positions |
| **Close-out** | Mandatory purchase to resolve a fail position |

## System Terms

| Term | Definition |
|------|------------|
| **Aging** | Number of days a fail has been outstanding |
| **Offset** | Matching a receive fail against a deliver fail |
| **Priority Score** | Calculated urgency based on multiple factors |
| **Escalation** | Routing to senior operations or compliance |

## Business Terms

| Term | Definition |
|------|------------|
| **Counterparty** | The other party to a trade |
| **CUSIP** | Committee on Uniform Securities Identification Procedures - security identifier |
| **Position** | Net quantity held in a security |
| **Recall** | Request to return borrowed securities |
